# Alexander's Developer Portfolio

## Projects

### 1. Wealth Dashboard

Live app: https://portfolio-tracker-vqap.onrender.com  
GitHub: https://github.com/alexbwight/portfolio-tracker

A privacy-first wealth dashboard for quick portfolio insights.

Features:
- Tracks crypto, stocks, and ETFs
- Uses live market prices
- No login or account linking
- Portfolio health score
- Future value projection
- Goal estimate
- Monthly investing estimate
- Quick insights
- Feedback collection through BuildPulse

Tech used:
- Python
- Flask
- yfinance
- HTML/CSS
- Render
- GitHub

What I learned:
- Building a full web app
- Working with APIs
- Deployment
- Git/GitHub
- UI design
- Product feedback
