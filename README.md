A simple, privacy-first wealth dashboard for quick portfolio insights.

## What it does

Wealth Dashboard lets users manually enter their crypto, stock, and ETF holdings, then instantly see a simple overview of their portfolio.

The goal is to make portfolio tracking feel simple, private, and beginner-friendly without requiring account linking.

## Features

- Track crypto, stocks, and ETF holdings
- No login required
- No broker or exchange account linking
- Live market prices using `yfinance`
- Portfolio health score
- Future value projection
- Goal estimate
- Monthly investing estimate
- Daily move estimate
- Quick portfolio insights
- Mobile-friendly responsive UI
- Feedback link powered by BuildPulse

## Why I built it

Most portfolio tools are either too complex, require account linking, or feel overwhelming for beginners.

This app is meant to be:

- simple
- private
- fast
- easy to understand
- useful for quick portfolio checks

## Connected Project

This app is connected to **BuildPulse**, a feedback collector app I built to collect structured user feedback.

Flow:

1. User opens Wealth Dashboard
2. User clicks **Leave feedback**
3. Feedback is submitted through BuildPulse
4. I can review feedback in a private BuildPulse dashboard
