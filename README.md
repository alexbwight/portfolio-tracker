# Wealth Dashboard

A simple no-login wealth dashboard for quick portfolio insights.

## Live App

https://portfolio-tracker-vqap.onrender.com

## Features

- Track crypto and stock holdings
- No login or account linking
- Portfolio health score
- Future value projection
- Goal estimate
- Monthly investing estimate
- Quick insights
- Feedback link

## Why I built it

Most portfolio tools are either too complex or require account linking.

This app is meant to be simple, private, and fast.

## Tech Stack

- Python
- Flask
- yfinance
- CoinGecko API
- Render
